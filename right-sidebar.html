Projects built from scratch

Wedding Guestlist

Created an application that displays a wedding guest list as more names are added or deleted. The default names include the bride and groom, and then new guests are stored using a Mongoose/MongoDB database. The front-end is created with HTML, CSS, JavaScript, and EJS.

Gym Exercises Library

Utilized React.js to create a webpage that displays exercises based on what the user searches for. The user may search for a specific body part, equipment required, or target muscle group. The results are generated from a RapidAPI Exercise database, and another YouTube Search and Download API is used to display video demonstrations of the exercise.

Power BI Project: Percent of Americans with Diabetes

Utilized Power BI to extract data from 6 different related tables from the CDC database on percentage of Americans with diabetes. Used Query Editor to merge tables and create reports to identify correlations and trends that demonstrate differences in diabetics' age, ethnicity, level of education, gender, and state.

Dog Library

Created an application where users can search for any dog breed and the app will populate data on that dog breed. The data includes ratings on different charactersistics typical to that dog breed and is pulled from an API database. Uses HTML, CSS, and JavaScript.
